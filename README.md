# Personal Portfolio Website

## Lab Exercise: Web Development Lab

### Project Description
This project is a personal portfolio website created as part of the **Web Development Lab** exercise. The purpose of this project is to design a simple and responsive portfolio website using HTML and CSS, and to practice Git and GitHub version control operations.

## Technologies Used

- HTML5
- CSS3
- Git
- GitHub
- Visual Studio Code

## Project Features

- Personal home page
- About me section
- Contact form
- Navigation menu
- Responsive design using CSS styling
- Organized project structure

## Website Pages

### 1. Home Page (index.html)
- Introduction about myself
- Welcome message
- Basic portfolio information

### 2. About Page (about.html)
- Personal information
- Educational background
- Skills section

### 3. Contact Page (contact.html)
- Contact form
- User input fields for name, email, and message
